### GET `/playlists/:id/contains/:map_id`

Checks if a playlist contains a particular map

#### Parameters

| Name   | Description            | Required |
|--------|------------------------|----------|
| id     | The id of the playlist | Yes      |
| map_id | The id of the map      | Yes      |

#### Example Response

```json
// https://api.quavergame.com/v2/10/20723

{
  "exists": true
}
```
