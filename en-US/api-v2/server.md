## Endpoints

### GET `/server/stats`

Gets the total user count, online users, score count, and mapsets on the server.

#### Parameters

None

#### Example Response

```json
// https://api.quavergame.com/v2/server/stats

{
  "online_users": 237,
  "total_mapsets": 33714,
  "total_scores": 53393773,
  "total_users": 689441
}
```
